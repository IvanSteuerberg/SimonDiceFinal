# SimonDiceFinal
Proyecto en AndroidStudio de un juego Simon Dice

Descripción:
4 botones de colores para el juego ,
3 butones para funcionalidades

Extras:
Puntuación ,
Idiomas inglés y español ,
Layout vertical y horizontal ,
Música ,
Selector de dificultad

Incluye el uso de:
ViewModel ,
Corrutinas ,
MediaPlayer
